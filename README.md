# BugFlow — Bug Tracking Tool

A modern, clean bug tracking dashboard built with React. Focuses on clarity and efficiency with crisp typography and an organized layout.

## Features

- **Dashboard** — High-level overview with stats cards, weekly activity chart, priority breakdown, and recently updated issues
- **Issues List** — Comprehensive table view with search, status/priority/type filtering, and color-coded badges
- **Issue Detail** — Full issue view with description, activity timeline, comments/discussion, and inline property editing
- **New Issue Modal** — Quick form to create bugs or feature requests with metadata

## Design

- **Fonts**: Syne (display), DM Sans (UI), JetBrains Mono (code/IDs)
- **Theme**: Dark, professional — deep navy palette with blue/violet/emerald accents
- **Charts**: Recharts for the activity bar chart

## Getting Started

```bash
npm install
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view in the browser.

## Build

```bash
npm run build
```

Builds the app for production to the `build` folder.

## Tech Stack

- React 18
- Recharts (charting)
- CSS custom properties (design tokens)
- Google Fonts (Syne, DM Sans, JetBrains Mono)
